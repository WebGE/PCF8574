Le circuit intégré PCF8574 : remote 8-bit I/O expander for I 2C-bus

  • Présentation

    • Le PCF8574 est un circuit intégré permettant de contrôler huit entrées / sorties numériques par l'intermédiaire d'un bus I²C. Il peut être utiliser pour interfacer des Led ou des boutons-poussoir comme dans l'exemple d'interface homme machine présentée ci-dessous.

  • Schéma interne du composant

  • Adressage

  • Documentation: pdf
  • Distributeur: Farnell

IHM SSI

Deux PCF8574 servent d'interface à l'Interface Homme Machine SSI qui se compose :

  • d'un afficheur I²C 2x16 (MIDAS ou BATRON) à circuit PCF2119,
  • de huit leds et de huit boutons-poussoir connectés à deux PCF8574.

Contenu du répertoire proposé au téléchargement

Le répertoire proposé au téléchargement (au format zip ou tar.gz) contient la solution NetduinoPCF8574 composée :

  • Du projet PCF8574 : le code source de la classe PCF8574.
  • Du projet TestNetduinoPCF8574 : un exemple d'utilisation de cette classe.

Remarques : La dernière version compilée de la classe PCF8574 se situe dans la bibliothèque MicroToolsKit disponible sur nuget.org.

Installer cette bibliothèque dans le projet simplifie l'utilisation de cette classe.

Description et utilisation de la classe PCF8574

  • Rôle: Contrôler huit entrées / sorties numériques.
  • Assembly: MicroToolsKit (disponible sur nuget.org)
  • Espace de noms: Microtoolskit.Hardware.IO

Création d'un projet avec l'IDE Visual Studio

  1. Utiliser le gestionnaire de paquets NuGet pour ajouter l'assembly MicroToolsKit (disponible sur nuget.org) dans le répertoire Références du projet.(clic droit sur Références puis Gérer les packages NuGet...)
  2. Ajouter l'espace de noms Microtoolskit.Hardware.IO dans l'en-tête du fichier source.
  3. Créer un objet en utilisant le constructeur PCF8574.
  4. Utiliser la méthode Write pour accéder au port en écriture et la méthode Read pour accéder au port en lecture.

Constructeur

Syntaxe Description
PCF8574(byte I2C_Add_7bits, Int16 Frequency) Instancie un objet "PCF8574(A)".
I2C_Add_7bits est à définir dans l'intervalle [0x20,0x27] pour un PCF8574 et dans l'intervalle [0x38,0x3F] pour un PCF8574A.
Frequency est à définir dans l'intervalle [100kHz,400kHz].

Méthodes publiques

Syntaxe Description
byte Read() Renvoie un octet représentatif de la valeur présente sur le port du PCF8574(A).
void Write(byte value)

value: octet à écrire sur le port du PCF8574(A).

PCF8574 : remote 8-bit I/O expander for I 2C-bus

  • Présentation

    • Le PCF8574 est un circuit intégré permettant de contrôler huit entrées / sorties numériques par l'intermédiaire d'un bus I2C. Il peut être utiliser pour interfacer des Led ou des boutons-poussoir comme dans l'exemple d'IHM ci-dessous.

Classe PCF8574

Contenu du dossier à télécharger

Le dossier à télécharger (au format zip ou tar.gz) contient le répertoire de la solution NetduinoI2CLEDBP composée :

  • Du projet PCF8574 : le code de la classe PCF8574.
  • Du projet TestNetduinoPCF8574 : un exemple d'utilisation de cette classe.

Utilisation

  1. Ajouter la référence PCF8574 et l'espace de nom ToolBoxes dans le projet.
  2. Créer un objet en utilisant un des constructeurs PCF8574.
  3. Utiliser la méthode Write pour accéder au port en écriture et la méthode Read pour accéder au port en lecture.

Constructeurs

Syntaxe Description
PCF8574(byte I2C_Add_7bits, Int16 Frequency) Instancie un objet "PCF8574(A)".
I2C_Add_7bits est à définir dans l'intervalle [0x20,0x2F] pour un PCF8574 et dans l'intervalle [0x38,0x3F] pour un PCF8574A.
Frequency est à définir dans l'intervalle [100kHz,400kHz].

Méthodes publiques

Syntaxe Description
byte Read() Renvoie un octet représentatif de la valeur présente sur le port du PCF8574(A).
void Write(byte value)

value: octet à écrire sur le port du PCF8574(A).
